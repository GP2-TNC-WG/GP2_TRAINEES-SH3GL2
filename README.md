### Summary
This is the online repository for the manuscript titled "___SH3GL2 variants and the risk for Parkinson‚Äôs disease in diverse ancestry populations___". This study focuses on exploring the role of SH3GL2 in existing PD/control datasets from the AMP-PD and GP2 initiatives.

### Data Statement
All data was using release 5 GP2 data (access-controlled via single-sign on amp-pd.org).
